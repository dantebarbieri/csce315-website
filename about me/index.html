Introduction to Programming

My very first programming language was BASIC. I often spent my time in my math classes teaching myself the Texas Instruments (TI) variant of BASIC (Language Reference) which ran on my TI-84 Calculator. Perhaps I should have spent more time paying attention to my math lessons, but I came away from my math courses having learned fundamentals of programming by writing random programs and testing unfamiliar instructions to see what worked. I wasn't familiar with the return statement, so I unfortunately made heavy use of GOTO statements. I've mostly broken that bad habit!

Science Fair

In Middle School and High School, I often competed in the Science Fair, presenting some technical experiment or another. This was one of my first projects where I used a laser and a receiver to count the number of cars that broke the path of the laser. A Python program I wrote would determine if the break in the laser was long enough to be a car and then would add 1 to the tally. Every couple of minutes or so, the program would output the number of cars into a file. The system was supposed to be a cheaper alternative than the pressure sensors built into roads to determine the volume of cars an intersection must support.
