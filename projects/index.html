3D Ball Bounce

Link to GitHub Repository

This is my attempt to simulate the 3D physics of a ball inside of a box. You can rotate the scene with the mouse and other keyboard controls are below:

LEFT Click ‐ Accelerate the ball vertically (think of it somewhat like dribbling in Basketball).
L ‐ Toggle unit velocity vectors for the ball.
K ‐ Toggle the shadow underneath the ball.
B ‐ Bounce the ball upwards.
WASD ‐ Move the ball forwards, left, backwards, and right, respectively.
Brackets ([/]) ‐ Decrease/Increase the impulse of the ball.
Plus/Minus (+/-) ‐ Zoom in/out (press and hold).

Traveling Salesman Bogo/Bozo Solver

Link to GitHub Repository

This solver attempts to find the shortest route between the 12 shown points. It could do this iteratively by checking every solution, but instead it randomly checks switching 2 paths, 3 paths, 4 paths, etc. all of the way up to randomly switching all 12 paths and testing the distance. This can more quickly approach a correct solution than can an iterative solver at the cost of not being able to halt and return a definitively shortest path.

Orbital Art

Link to Web Hosting

This physics simulation has small objects with mass and larger orbiting objects that are massless. The massless orbiters feel the effect of the primaries (objects around which other objects orbit) and choose which primary to orbit based on Newtonian Universal Gravitation. Controls:

LEFT Click ‐ Create a new primary.
C ‐ Clear the canvas of the art.
A ‐ Toggle auto-clearing of the art.
X ‐ Remove the most recently added primary.
Plus/Minus (+/-) ‐ Add or remove an orbiter.
Brackets ([/]) ‐ Change the length of the ring trials (Warning: Long trails really hurt performance!).
M ‐ Toggle visibility of primaries.
